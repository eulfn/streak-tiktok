## What's new in this release (v4.0.3)

### Hotfix: Group Automation

- **Group Messaging Fixed**: Resolved a critical issue where the automation service would get stuck on the page rendering phase when attempting to process group chats. This was caused by an internal null reference preventing the automation script from injecting.
- **Enhanced Group Detection**: Improved the web script's ability to identify group chats on the newer TikTok desktop web layout, preventing group chats from being incorrectly skipped as DMs.
- **Log Improvements**: The app logs now correctly display the group name (e.g., `[PROCESS] [Group: My Group]`) instead of a blank `[@]` when starting automation for a group.
