## v1.9.2 (Structural Fix)

This release addresses critical structural XAML compilation errors and duplicate resource keys that were causing build failures in the previous versions.

### Fixes

*   **XAML Structural Integrity**: Restored the proper <Setter> wrapping for all VisualStateManager.VisualStateGroups blocks across Styles.xaml.
*   **Resource Cleanup**: Removed a duplicate BorderColorLightBrush key from Colors.xaml that triggered compilation errors.
*   **Build Stability**: Verified the entire XAML codebase with xmllint to ensure 100% compliance with XAML schema requirements.
